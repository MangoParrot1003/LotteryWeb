========================================
🎲 学生抽签系统 - 快速开始指南
========================================

📌 3步启动系统：

第1步：启动后端
   👉 双击运行：start-backend.bat
   ⏳ 等待看到：Now listening on: http://localhost:5000

第2步：启动前端
   👉 双击运行：start-frontend.bat
   ⏳ 首次运行会自动安装依赖（需要2-5分钟）
   ⏳ 等待看到：Local: http://localhost:5173

第3步：访问系统
   👉 打开浏览器
   👉 访问：http://localhost:5173
   🎉 开始使用抽签系统！

========================================
💡 使用提示：

✨ 基本抽签：点击"开始抽签"按钮
🎯 性别筛选：选择男/女，只从该性别中抽取
📚 班级筛选：选择班级，只从该班级中抽取
📊 查看统计：页面顶部显示学生总数

运行 test-api.bat 可以测试所有 API 接口

========================================
❓ 遇到问题？

1. 确保已安装 .NET 8.0 SDK
2. 确保已安装 Node.js 18+
3. 检查端口 5000 和 5173 是否被占用
4. 查看"启动检查清单.md"获取详细帮助
